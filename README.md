- Pip (voor Python pakketbeheer)

### Stappen om te installeren
1. Clone de repository naar je lokale machine:

    ```bash
    git clone https://github.com/jouwgebruikersnaam/jouwprojectnaam.git
    ```

2. Ga naar de projectmap:

    ```bash
    cd jouwprojectnaam
    ```

3. Installeer de benodigde pakketten:

    Voor Python-projecten:

    ```bash
    pip install -r requirements.txt
    ```

    Voor Node.js-projecten:

    ```bash
    npm install
    ```
